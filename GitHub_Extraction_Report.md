# Databricks notebook source
# MAGIC %md This notebook sets up the companion cluster(s) to run the solution 
# accelerator. It also creates the Workflow to illustrate the order of execution.
# 
# Steps:
# 1. Simply attach this notebook to a cluster and hit `Run all`.
#    A multi-step job and the clusters used in the job will be created for you.
#    Hyperlinks are printed on the last block.
#
# 2. Run the accelerator notebooks:
#    2a. Run the Workflow: Navigate to Workflow link, hit 'Run Now'

# Prerequisites:
# 1. Cluster creation permissions in this workspace
# 2. In case of cluster-policies, manually create per the job_json config

# Notes:
# 1. Pipelines, workflows and clusters are NOT user-specific.
#    Rerunning resets them for all users.
